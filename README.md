OWASP ZAP Vulnerability Assessment Lab

Objective

The objective of this lab was to install and use OWASP ZAP to perform a vulnerability assessment against OWASP Juice Shop and identify common web application security weaknesses.

Tools Used

- Kali Linux
- OWASP ZAP 2.17.0
- OWASP Juice Shop
- VirtualBox

Environment

- Attacker Machine: Kali Linux
- Target Application: OWASP Juice Shop
- Scanner: OWASP ZAP

Procedure

1. Installed OWASP ZAP on Kali Linux.
2. Verified Java installation.
3. Launched OWASP Juice Shop.
4. Started OWASP ZAP.
5. Performed automated vulnerability scanning.
6. Reviewed alerts and findings generated by ZAP.

Findings

The scan identified several security findings including:

- Content Security Policy Issues
- Cross-Domain Misconfiguration
- Missing Anti-Clickjacking Header
- Session ID in URL Rewrite
- Vulnerable JavaScript Library
- Information Disclosure Findings

Results

OWASP ZAP successfully scanned OWASP Juice Shop and identified multiple web application security weaknesses. The lab demonstrated the use of automated vulnerability assessment tools to discover security issues within a web application.
